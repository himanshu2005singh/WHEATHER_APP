A sleek, responsive web application that provides real-time weather data for cities worldwide. This project demonstrates my ability to integrate Third-Party APIs, handle Asynchronous JavaScript, and create a clean User Experience (UX).

Key Features
Real-Time Data: Fetches live weather conditions (temperature, humidity, wind speed) via the OpenWeatherMap API.

Dynamic UI: Backgrounds or icons change based on the current weather condition (Clear, Rain, Snow, etc.).

Search Functionality: Instant weather updates for any city globally.

Geolocation: (Optional: include if you used it) Automatically detects the user's current location for immediate local updates.

Responsive Design: Fully optimized for mobile, tablet, and desktop screens.

Tech Stack
Frontend: HTML5, CSS3 (Flexbox/Grid), JavaScript (ES6+)

API: OpenWeatherMap API

Deployment: GitHub Pages / Netlify
What I Learned
Asynchronous Programming: Mastered the use of fetch() and async/await to handle API calls and JSON data.

Error Handling: Implemented robust logic to catch invalid city names or network failures without crashing the UI.

DOM Manipulation: Dynamically updating the interface in real-time based on user input.
